Homework 2: Datasets and Distributions

Exploring the fundamental concepts of datasets and probability distributions through practical exercises using Microsoft Access and basic statistical computation.

1. Concept of a Dataset

A dataset is a structured collection of data, typically represented in tabular form, where each row corresponds to an observation (or record) and each column corresponds to a variable (or feature). In the context of cybersecurity, datasets often contain network logs, user activity, or system events used for threat detection and anomaly analysis.

Each dataset must define clear attributes (such as IP address, timestamp, packet size) and a domain of values. A well-designed dataset enables descriptive statistics, visualization, and inference.

2. Concept of Distribution

A distribution describes how values of a variable are spread or concentrated across possible outcomes. In statistics, distributions allow us to quantify likelihoods, detect patterns, and model uncertainty. Common examples include the normal distribution, uniform distribution, and Poisson distribution.

In cybersecurity, understanding distributions helps identify anomalies — for instance, unusually high packet transmission rates might indicate a Denial-of-Service (DoS) attack if they deviate significantly from the expected statistical distribution of network activity.

3. Creating a Dataset using a DBMS (Microsoft Access)

For this exercise, a small dataset was created using Microsoft Access, simulating a simplified log of cybersecurity events. The table contains the following fields:

  • Event_ID – Primary key (integer)
  • User – Identifier of the system user (text)
  • Login_Attempts – Number of login attempts (integer)
  • Bytes_Sent – Data transmitted (integer)
  • Event_Type – Type of activity (categorical: “Normal”, “Suspicious”, “Attack”)

4. Computing the K Univariate Distributions (K ≤ 3)

We computed three univariate distributions using the dataset above:

  1. Login_Attempts Distribution: Frequency of login attempts among users.
  2. Bytes_Sent Distribution: Distribution of transmitted data volume.
  3. Event_Type Distribution: Categorical frequency of event types.
